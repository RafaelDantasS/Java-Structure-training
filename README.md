# Java-Structure-training
Projeto desenvolvido para melhorar meu conhecimento em conceitos da programação e na linguagem Java

## Proposta do projeto
Aperfeiçoar minha base da programação assim como entender melhor como as biblioteas que frequentamente utilizamos funcionam. Além disso, sempre que o projeto atingir um certo nível de maturidade, o intuito é começar a aplicar novos conceitos e documentar todas as mudanças adicionadas de forma a explicar o que foi alterado assim como o motivo de ter tomado tal abordagem.

## Problema a ser trabalhado 
O problema em questão é até que simples, basicamente seria uma forma de aperfeiçoar o sistema de um restaurante de forma que organize os clientes por quantidades de pessoas no mesmo grupo assim como em qual mesa melhor se adequa. além disso tambem lidar com uma lista de espera cujo o programa deve alocar os clientes que melhor se adequam a mesa que esta disponivel no momento.
